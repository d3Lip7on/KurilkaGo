В этой директории создаете папки своих компонентов, например "header", "footer", "slider" и тп... 
Ваша папка компонента должна состоять из html, css и js файлов, содержащие код ТОЛЬКО вашего компонента. Также предусматривается папка images для картинок.
Пример структуры папки компонента и кода показан в директории test-button
